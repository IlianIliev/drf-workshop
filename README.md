## Project setup
 1. Adding `requirements.txt`
    - Django==2.1.1
    - psycopg2==2.7.5
 2. Creating Django project
    - django-admin startproject drf_workshop .
 3. Configuring DATABASES
 4. Run the project:
    - `./manage.py runserver`
